CANCELAMENTOS

Cancelamentos - Consultas Especializadas:

Os cancelamentos poderão ser realizados pelo assistente administrativo da unidade saúde via sistema Vivver.

Obs.: Manter a regulação da guia antes de cancelar.

Para cancelamentos com necessidade de remarcação, encaminhar solicitação por e-mail para a Central de marcação de consultas com justificativa para o cancelamento. A CMC ficará responsável de inserir o paciente na fila de espera.

Retornos CCE Ressaca:

  • Agendamentos via WhatsApp
  • CISMEP/CCE Ressaca: inserir no sistema

Retornos Médicos Fora da Rede:

Encaminhar e-mail para a Central de Marcação de consultas inserir no sistema.

Devolução de Documentos:

Encaminhar para o CEAPS para que o órgão responsável possa realizar a devolutiva.

PROCEDIMENTOS ESPECÍFICOS

PRÉ-NATAL DE ALTO RISCO

• Profissional de nível superior insere solicitação com justificativa no sistema Vivver

• Arquivar guia na unidade para monitoramento

• UBS acompanha via Sistema Vivver verificando pendências

• UBS recebe relatório dos agendamentos via malote

• Administrativo imprime, anexa à guia e entrega ao paciente

Pendências: Verificar diariamente - só profissionais de nível superior podem corrigir

SANGRAMENTO UTERINO

• Profissional de nível superior insere solicitação com justificativa

• Arquivar guia na unidade para monitoramento

• UBS acompanha via Sistema Vivver verificando pendências

ACUPUNTURA

• Profissional de nível superior insere solicitação

• Arquivar guia na unidade para monitoramento

• UBS recebe relatório via malote

• Administrativo entrega agendamento ao paciente

Ressalva: Solicitações de profissionais externos podem ser inseridas pelo administrativo, anexando a guia para regulação

FISIOTERAPIA

• Agendamentos pela UBS via Sistema Vivver

• Pode agendar para outros distritos se sem vagas

• Paciente deve concordar com local

Buco-Maxilo: Agendar para clínica Fisiovida

Geniturinária (Pélvica): Encaminhar para setor de reabilitação - atendimentos no CCE Iria Diniz
